유국 처리 규정

EXHAUSTIVE DRAW
  1. 70순 유국 — 70순의 패가 모두 버려진 경우 유국. 친이 텐파이 상태라면 친은 흘러가지 않는다.
  2. 4인 리치 유국 — 마지막 리치 선언패의 통과를 확인하고 유국한다.
    • 텐파이가 아닌 사람이 있다면 촌보 처리
  3. 4깡 유국 — 두 명 이상의 대국자가 깡을 총 4번 선언한 경우 최종 타패 후 유국.
    • 한 명의 대국자가 4번 깡을 선언한 경우, 4번째 이후 깡 선언 불가 / 대국 속행
  4. 구종구패 — 유국
  5. 사풍연타 — 유국 (친은 넘어가지 않음)
  • 4인리치 / 4깡 / 구종구패는 친이 넘어가지 않는다.
  • 트리플론의 경우 친이 쏘일 시 친은 넘어간다.
  • 촌보를 제외한 모든 유국은 연장이 증가한다.
05

촌보 처리 규정

CHOMBO
패널티 · 게임 중 벌칙으로 촌보한 경우, 촌보자는 다른 플레이어 모두에게 각 3,000점의 점봉을 지불하고 해당 게임을 무효화한다.
  • 친은 절대 흘러가지 않으며, 오라스라도 최종국을 재시작 (단, 촌보로 들통된 경우는 게임 종료)
  • 더블론에서 패 공개 후 촌보 발생 시 — 촌보하지 않은 한 명의 화료를 인정하고, 다른 한 명에게는 촌보 패널티를 부여하지 않는다.

대상 역만 · 대삼원 / 대사희 / 사깡즈 / 사리엔커

벌칙(패널티)

BGM · BOARDGAME & MAHJONG
09

패널티 분류

PENALTY TYPES
LV.1

1천봉
공탁

  • 발성 미스 (치 / 펑 / 깡)
  • 발성 후 소·다패 확인
    • 발성 직후 = 가져온 패 되돌림
    • 타패 후 확인 = 공탁 없이 속행
  • 패산을 잘못 가져온 경우 (되돌린 후 공탁)
  • 패산 잘못 + 타패 완료 = 공탁 후 속행
LV.2

화료
불가

  • 화료 관련 발성 미스 (쯔모 / 론)
  • 소·다패
  • 패산 잘못 + 타패 완료 + 旣 리치 대국자 존재 = 실수자 화료 불가
  • 리치 후 화료 발성 실수 = 유국 시 텐파이 확인 후 노텐 처리
LV.3

촌보

  • 화료 발성, 공개 후 화료가 안 된 경우
  • 소·다패 확인 후 고의 발성
  • 한번에 4개 이상 패 공개 (본인·타인·패산 모두)
  • 다른 대국자의 패를 보는 경우
  • 노텐 리치 시 유국이 된 경우
  • 후리텐 시 론을 한 경우
  • 리치 후 허용되지 않은 깡을 한 경우
  • 발성 후 패를 가져오지 않고 유국 후 확인
10

기타 운영 원칙

PRINCIPLES
  • 배패되거나 진행 중인 패는 무조건 플레이가 되어야 한다. 다시 섞어야 하는 경우 상황의 잘잘못에 따라 촌보 처리한다. (대회 시에만 적용)
  • 오라스 점수는 정확히 계산해야 한다. 잘못 알려주어 문제가 생길 경우, 책임은 잘못 불러준 개인이 아닌 해당 게임에 참여한 모든 플레이어에게 있다.
  • 쯔모를 위해 다음 패를 잡은 경우, 패를 만진 사람은 쯔모를 취소하고 치 / 펑 / 깡 등의 후로를 선언할 수 없다. (론화료는 가능)
※ 초보자 / 미숙자 배려 · 패널티는 최소한으로 배려해주되, 상황에 따라 심각하거나 실수가 반복되는 경우 해당 자리에 있는 멘토가 판단할 수 있다. 대회의 경우 다소 엄하게 적용한다.
